# 🔥 Phoenix Terminal v3

Cyberpunk trading terminal for Phoenix perpetuals on Solana.

**Web-only product** — connect your wallet, configure the swarm bot, and trade entirely through the browser.

## Quick Start

```bash
cd web
npm install --legacy-peer-deps
cp .env.example .env.local
npm run dev
```

Open http://localhost:3000 and connect your Solana wallet.

See [web/README.md](web/README.md) for full documentation.
